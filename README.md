# Belivia 2.0

Belivia 2.0 is a custom website, backend and infrastructure project for a real client in the area of everyday assistance for seniors and relatives.

## Project scope

This repository is used for:

- project documentation
- frontend
- backend
- admin/ops panel
- deployment configuration
- infrastructure-related files

## Planned stack

- Ubuntu 24.04 LTS
- Hetzner Cloud
- Cloudflare
- Cloudflare Tunnel
- Nginx
- Static frontend
- FastAPI
- SQLite
- systemd
- Tailscale

## Repository structure

- `docs/` project documentation
- `frontend/` public website
- `backend/` public API/backend
- `admin/` internal admin/ops application
- `deploy/` deployment and server-related config
- `assets/` branding and content assets
- `local/` local helper scripts and notes
